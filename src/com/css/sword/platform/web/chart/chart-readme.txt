关于SWORD CHART使用说明：
例子：
demo目录下demo/LineStyle.java和assemblyChartTest.java
可以通过http://localhost:Port/ContextName/sword?tid=ChartDemo1BLH_init来看demo。
API使用:
可以通过ant来构建生成关于图表的使用API或者其它;当期目录下已经提供了build.xml来构建。
(需要把build.xml文件拷贝到工程的根目录下即可。生成dist/chart-docs/)
